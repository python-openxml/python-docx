.. _WdCellVerticalAlignment:

``WD_CELL_VERTICAL_ALIGNMENT``
==============================

.. autoclass:: WD_CELL_VERTICAL_ALIGNMENT

   .. attribute:: TOP
      :annotation:

      Text is aligned to the top border of the cell.

   .. attribute:: CENTER
      :annotation:

      Text is aligned to the center of the cell.

   .. attribute:: BOTTOM
      :annotation:

      Text is aligned to the bottom border of the cell.

   .. attribute:: BOTH
      :annotation:

      This is an option in the OpenXml spec, but not in Word itself.
      It's not clear what Word behavior this setting produces. If you
      find out please let us know and we'll update this documentation.
      Otherwise, probably best to avoid this option.
